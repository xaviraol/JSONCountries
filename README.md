# JSONCountries

Basic countries information in json and xml format.

Each country containts the following information:
- country_name
- country_code (key)
- country_region
- country_capital 
- country_languages
- currency_name
- currency_code
- currency_symbol


It combines information from:
https://gist.github.com/Fluidbyte/2973986
https://github.com/mledoze/countries
